# 简历写作技法库

> 这些是进刀六步之外的**写作技法**——不是规则，是让简历从"合格"到"锐利"的手感。在进刀过程中按需使用。

## 概要段三段式

概要段不是自我介绍，是HR的10秒锚点。三句话，每句有明确功能：

1. **领域锚点**：你在哪几个行业/领域做什么——让HR判断你跟岗位匹不匹配
2. **可验证战绩**：你干过的最硬的1-2件事，带数字——让HR判断你行不行
3. **差异化标签**：你跟同类候选人不同的地方——让HR记住你

> 反面：'跨界复合型操盘手，具备卓越的商业化能力'——三个形容词，零信息。
> 正面：'做过金融、网安、AI三个行业的B2B市场，擅长把技术产品变成客户能买单的方案。零资源起步4个月新人第一，在央企从零跑通首单海外直营。'

**概要双格式：** 段落式（上例）之外，可选用编号主题式，适用于跨行业经历多、能力维度丰富的候选人：

```
1. 复合型B2B商业化能力：跨越X个行业，始终聚焦一件事——[核心能力的一句话定义]。
2. 战略与执行双重经验：既能[上层能力]，也能[一线能力]。[最硬战绩+数字]。
3. 复杂问题解决与流程沉淀：经手[具体复杂场景]，均沉淀为可复用的SOP——具备把踩坑经验转化为组织资产的能力。
```

选择标准：经历跨3个以上行业→优先编号主题式（更显迁移能力）；经历聚焦1-2个行业→段落式（更直接）。

## 数字前置法

每条经历里最硬的数字放到句首或紧贴小标题。HR扫读时只看每条的前半句，数字在后半句等于没有。

> 反面：'负责海外客户拓展，历时6个月跟进，通过多轮技术方案汇报，最终落地合同额40万美金。'
> 正面：'合同额40万美金。从零跟进海外客户6个月，直接对接VP及运营总监完成多轮技术方案汇报。'

## 边界条件配缀法

每个强动词后面紧跟一个限定短语，说清"管到哪一层"。这是庖丁最核心的技法——不是删动词，是给动词配定语。

配缀三要素（至少含其一）：
- **环节限定**：管了哪个环节（需求对接 / 功能验收 / 方案设计）
- **层级限定**：管到哪一级（向VP汇报 / 协同财务复核 / 牵头IT部门）
- **维度限定**：用什么维度做的（按商机/续费/渠道三维拆分 / 分三批迭代上线）

> '主导IT系统建设' → 没有配缀，HR追问'你一个市场岗凭什么主导IT'
> '主导与IT部门的需求对接与功能验收' → 有环节限定+层级限定，经得起追问

**边界条件三层升级：** 配缀不只是"你管到哪一层"，还要交代两个额外维度：

1. **交付对象层级**：这份产出最终呈现给谁看？
   > '主笔十五五规划' → '主笔十五五规划的对标模型与财务数据测算，材料直接提交集团战略委员会'
2. **具体判定阈值**：你的规则/流程里有没有可量化的触发条件？
   > '设定风险预警机制' → '设定关键风险预警阈值（如决策链变动、测试延期超5天）'
3. **唯一责任人信号**：你是不是这件事的单一接口人？
   > '负责产研对接' → '作为业务侧唯一接口人对研发排期'

三层都做到=每一次动词都经得起面试官的连续追问。

## 职业主线缝合

跨行业简历最大的问题是"散"。每段经历看起来都不错，但读完不知道这人的主线是什么。解法：在概要段用一句话把所有经历串起来。

> 缝合前：金融→网安→AI→汽车，四段独立经历，HR困惑'你到底深耕哪个行业'
> 缝合后：'6年聚焦B2B复杂解决方案的市场化——不管在金融、网安还是汽车，我干的都是同一件事：把技术翻译成商业语言。'

这句话把跨行业从"缺点"变成了"迁移能力"。

## 具体案例故事节奏

在关键经历中加入一个"发现问题→定位→解决→量化"的微故事，让HR能想象你工作的样子。不是每条都要，选最硬的1-2条即可。

节奏模板：
```
[做了什么，说清背景]。[曾发现/遇到一个具体问题]。[怎么定位和解决的]。[带来的量化改变]。
```

> 空洞版：'从零搭建经营分析体系，按月输出多维数据报告。'
> 故事版：'从零搭建月度经营分析看板，按商机、续费、渠道贡献等维度拆分数据。曾发现某产品线毛利率异常，追溯定位原因后协助总经理调整定价与渠道返点策略，经营复盘会决策效率提升约1倍。'

四点节奏让HR能"看见"你的分析能力和业务影响力，而不是只能"相信"你的简历标题。

## 定量对比替代形容词

任何"大幅""显著""有效"都可以替换为定量对比。没有精确数字时用定性对比：

| 形容词（删） | 定量对比（留） |
|---|---|
| 大幅提升 | 从2h压缩到40min，效率翻倍 |
| 显著改善 | 回款周期缩短约30% |
| 有效支撑 | 发放超3000册，销售用于客户拜访 |
| 成功落地 | 首单，此前集团无先例 |

## 动词通胀降级表

当一段经历里强动词过于密集时，按实际参与深度逐级降档：

| 原文动词 | 降一档 | 降两档 | 什么时候降 |
|---|---|---|---|
| 主导 | 主笔/牵头 | 负责 | 上面有老板拍板时降一档，只做了某一块时降两档 |
| 操盘 | 独立负责 | 执行 | 同上 |
| 统筹 | 协调 | 协助 | 同上 |
| 全方位 | 全流程 | — | "全方位"永远删除，替换为"全流程" |
