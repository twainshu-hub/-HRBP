# 庖丁 · 简历打磨师

> 刀刃只在骨缝间走。

庖丁是一个 WorkBuddy Skill，用于简历打磨。把 AI 写简历的四类硬伤变成可诊断、可修复、可验证的标准流程。

## 设计哲学

"彼节者有间，而刀刃者无厚"——不改不该改的，不砍不该砍的。不是往里塞东西，是找简历里本来就有的骨头。

市面上改简历的工具都是往里加东西——加动词、加数字、加包装。但一份简历真正的毛病往往不是"少了什么"，是"多了什么"。

## 四道工序

1. **辨骨** — 四类硬伤诊断（量化阈值 + 优先级）
2. **寻隙** — 四档动词校准表
3. **进刀** — 六步 + 简历写作技法库
4. **收刀** — L1-L4 四层自检

## 使用方式

在 WorkBuddy 中安装本 Skill 后，直接说出"帮我改简历"或"庖丁"即可触发。
